# Dubai Real Estate Data Analysis

Exploratory data analysis of the Dubai rental market using Python, uncovering pricing trends across property types, locations, and size metrics.

## Overview

This project analyses a dataset of Dubai rental listings to answer key questions:
- Which locations are the most expensive and most affordable?
- Which property types command the highest rents?
- What factors most strongly predict rent price?
- Where is land value highest per sqft?

## Key Findings

- **Most expensive location:** Emirates Hills (AED 2.5M median rent)
- **Most affordable locations:** Al Mujarrah, Rolla Area, Al Nabba
- **Highest value per sqft:** Bluewaters Island (AED 315/sqft)
- **Strongest rent predictors:** Property area (0.41) and bedroom count (0.31)
- **Most expensive property type:** Residential Floor (AED 1.75M median rent)

## Visualisations

- Average rent by property type
- Top 10 most expensive locations
- Top 10 most affordable locations
- Top 10 most valuable locations per sqft
- Rent per sqft distribution
- Correlation heatmap of key numeric variables

## Tech Stack

- **Python** — data analysis
- **Pandas** — data manipulation and aggregation
- **Matplotlib / Seaborn** — visualisation
